This directory contains the code to reproduce the results in the paper, Beyond
time-homogeneity for continuous-time multistate Markov models.  
Please contact Emmett B. Kendall at ebkendal@ncsu.edu for any help or questions.

See workflow.sh for line-by-line Unix command line code for reproducing the results.

See packages.txt for a detailed list of all R packages (and their respective
versions) used in running the code.

The R files in real_ecog_analysis/ use a data set that has already been cleaned
and formatted for the purpose of these analyses. To find the raw data from the 
mice sleep study, please see the following URL:
https://archive.norstore.no/pages/public/datasetDetail.jsf?id=10.11582/2022.00033
